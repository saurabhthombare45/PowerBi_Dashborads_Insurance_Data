# Insurance Data Analytics Dashboard (Power BI)

## Overview

This project presents an **interactive Insurance Data Dashboard** built using Power BI. The dashboard analyzes insurance data to provide insights into policy performance, claims trends, customer demographics, and financial metrics. It enables stakeholders to make informed decisions through clear and interactive visualizations.

---

## Objectives

* Analyze overall insurance policy performance
* Understand claims patterns and settlement status
* Explore customer demographics and behavior
* Evaluate financial metrics such as premium and claims
* Identify trends using time-based analysis

---

## Dashboard Insights

### 1. Policy Overview

* Total number of policies
* Active vs expired policies
* Policy distribution by type
* Total premium collected

---

### 2. Claims Analysis

* Total claims filed
* Claim status (Approved, Pending, Rejected)
* Total and average claim amount
* Claim trends over time

---

### 3. Customer Demographics

* Gender distribution
* Age group analysis
* Customer segmentation based on policy type

---

### 4. Financial Analysis

* Total premium collected
* Total claim amount
* Comparison between premium and claims
* Coverage amount insights

---

### 5. Time-Based Analysis

* Policy issuance trends
* Claim trends over time
* Seasonal patterns in insurance activity

---

## Features

* Interactive slicers and filters (Policy Type, Gender, Claim Status, Date)
* Drill-down and drill-through capabilities
* User-friendly and visually structured layout
* Dynamic data exploration

---

## Tools & Technologies

* Power BI
* Power Query (Data Cleaning & Transformation)
* DAX (Data Analysis Expressions)

---

## Dataset

The dataset used in this project includes:

* Policy details (Policy Number, Policy Type, Start & End Date)
* Customer information (Customer ID, Age, Gender)
* Financial data (Premium Amount, Coverage Amount)
* Claims data (Claim Number, Claim Amount, Claim Status)

---

## How to Use

1. Open the `.pbix` file in Power BI Desktop
2. Navigate through dashboard pages
3. Use filters and slicers to explore specific insights
4. Interact with visuals for detailed analysis

---

## Conclusion

This dashboard provides a comprehensive view of insurance data by integrating policy, claims, customer, and financial analysis. It helps in identifying trends, improving risk assessment, and supporting better business decisions.
